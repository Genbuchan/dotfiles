## インストール方法
```shell
$ curl -L dotfiles.genbu.studio/$(uname) | bash
```

## 動作環境

### macOS
- macOS 10.15

### Linux
- Debian GNU/Linux 10 Buster
- Ubuntu 20.04 LTS
